## Running

Clone the repository and run the game

```bash
git clone https://github.com/moontr3/flappy-bird.git
cd flappy-bird
python main.pyw
```

## Controls

| Key | Action |
|-----|-----|
| `Space` / `LMB` | Jump/leap or start the game |
| `F3` | Show debug information |

_To restart reopen the program_
